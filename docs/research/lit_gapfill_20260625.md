- **Nitrobenzene:** No clean PRIMARY blank-window row confirmable. Primary CVs using nitrobenzene as solvent exist (Allen/Compton-type Br⁻/Br₂ work: *Electrochim. - **THF (tetrahydrofuran):** No verified numeric anodic+cathodic pair from a rule-compliant primary. Compton-group THF methods papers DO use a genuine Fc/Fc⁺ ref (sealed Fc/Fc⁺ + TBAPF₆ in THF) and TBAPF₆ electrolyte — rule-3/4 compliant: Paddon & Compton, *Electroanalysis* 2005, 17, 1919–1923 (10.1002/elan.200503326); Baron et al., *J. - **Sulfolane (TMS):** No clean rule-compliant PRIMARY blank window. Battery-grade anodic-stability data exists (Li-salt sulfolane electrolytes stable to ~5.5–5.9 V vs Li/Li⁺; *Russ. J. Appl. Ferrocene-in-sulfolane kinetics (*J. Solution Chem.*, 10.1007/s10953-006-9119-9) and sulfolane benzaldehyde voltammetry (*Anal. The single clean row (DMSO) came from an open-access RSC HTML article. **Recommended next step:** institutional / SciFinder / Reaxys full-text access to the cited candidate papers.
